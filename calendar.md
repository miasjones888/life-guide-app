## Design Principles

- Google Calendar is for alarms and time-anchored reminders ONLY
- It is not for task management or planning
- The paper system (notebooks, folders, index cards) handles task management
- The calendar handles: what fires when, what is due, and what is recurring
- Tasks are modular — everything except cat meds, personal meds, and financial deadlines can be rescheduled

## Calendar Event Descriptions — Key Content

### Weekly Setup (Sunday 7pm)
1. Review the week ahead in Google Calendar
2. Choose one anchor task per project folder
3. Write three weekly intentions in main notebook
4. Check the grocery list for anything missing
5. Write anchor task for Monday in the notebook

### Deep Focus Sessions
- Phone down. No email. No browsing. No switching.
- Anything that comes up goes on an index card
- 90-120 minutes. Then stop.
- Project folder must be open

### Morning Routine (7:30am)
1. Small notebook — morning pages, 20 mins
2. 5-minute sit / meditation
3. Reading before screens (even 10 mins)
4. Morning skincare (cleanser + SPF)
5. Cat morning meds + wet food breakfast
6. Eat something
No scrolling Instagram or TikTok until this sequence is done.

### Evening Routine (9pm)
1. Cat evening meds + dinner + litter
2. Bedtime meds (9:30pm)
3. Night skincare (cleanser + moisturiser)
4. Write tomorrow's anchor task in main notebook
5. Reading before sleep
No scrolling after this point. Lights out by 11pm.

## Notes on Modularity

All calendar blocks except the following can be rescheduled when life intervenes:

**Non-negotiable (do not move):**
- 🐱 Cat Morning Meds (9am daily)
- 🐱 Cat Evening Meds + Dinner + Litter (9pm daily)
- 💊 Your Bedtime Meds (9:30pm daily)
- Any financial deadline with a hard date (taxes, bill due dates)

**Modular (can shift):**
- Deep Focus sessions — move to any free 2-hour window
- Weekly Setup — can happen Monday if Sunday is unavailable
- Outside Time — can move to any weekday
- Self-Care Block — move within the week
- Laundry — any day of the week
- Room Reset — skip entirely if needed, resume next cycle
